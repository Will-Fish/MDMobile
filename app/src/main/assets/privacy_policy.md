# 鱼尾猩.md 隐私政策

感谢您使用鱼尾猩.md。本应用是一个以本地文件为核心的 Markdown 阅读与编辑器。

## 1. 本地存储的数据

应用仅在您的设备本地保存以下内容：

- 最近打开的文件记录
- 书签与阅读进度
- 主题、字号、默认目录等偏好设置

这些数据不会自动上传到任何服务器。

## 2. 权限用途

应用可能会请求文件访问相关权限，用于：

- 浏览设备中的 Markdown 文件
- 读取、编辑和保存文档
- 选择图片并插入到文档中
- 导出或分享当前文件

## 3. 数据共享

应用不会主动收集、出售或分享您的个人数据。只有在您主动使用系统分享功能时，系统才会按您的选择把文件发送给其他应用。

## 4. 数据安全

您的阅读记录、书签和设置保存在设备本地。请妥善保管您的设备与文件内容。

## 5. 联系方式

如果您对本隐私政策有疑问，可以通过应用相关项目页面或维护者提供的联系方式反馈。
